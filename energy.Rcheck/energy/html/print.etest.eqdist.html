print.etest.eqdist {energy}R Documentation

Print Multisample E-test (Energy Test) for Equal Distributions

Description

Print method for etest.eqdist object returned by the eqdist.test function.

Usage

print.etest.eqdist(x, ...)

Arguments

x an object of class etest.eqdist
... extra arguments passed to or from other methods

See Also

eqdist.etest

Examples

## print test if the 3 varieties of iris data (d=4) have equal distributions
 data(iris)
 e <- eqdist.etest(iris[,1:4], c(50,50,50))
 print.etest.eqdist(e)
