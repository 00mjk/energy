print.etest.poisson {energy}R Documentation

Print Mean Distance Test for Poisson Distribution

Description

Print method for etest.poisson object returned by the poisson.mtest function.

Usage

print.etest.poisson(x, ...)

Arguments

x an object of class etest.poisson
... extra arguments passed to or from other methods

See Also

poisson.mtest

Examples

 x <- rpois(20, 1)
 e <- poisson.mtest(x)
 print.etest.poisson(e)
